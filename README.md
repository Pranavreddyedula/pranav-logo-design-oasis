# PRANAV – Logo Design (KISS Principle)

This project is a minimalist logo design created as part of a graphic design exercise following the **KISS (Keep It Simple, Stupid)** principle.

## Objective
To design a clean, elegant, and memorable logo that represents a personal brand using minimal elements.

## Design Concept
- Monogram-based logo (PR)
- Gold color symbolizing elegance and value
- Dark green background representing trust and stability
- Simple typography with strong visual identity

## Tools Used
- HTML5
- CSS3

## Key Design Principle
KISS – Keep It Simple, Stupid

## Outcome
A professional, scalable, and modern logo design suitable for digital branding.
